# @microblink/unocss-config

## 0.1.0

### Minor Changes

- Adds vertical breakpoints and orientation variants for responsive height-based layouts. Includes `h-(xs|sm|md|lg|xl|2xl):` min-height variants, `lt-h-*:` max-height variants, `at-h-*:` exact range variants, `portrait:` / `landscape:` orientation variants, and dynamic viewport height utilities (`h-{num}dvh`, `h-{num}lvh`, `h-{num}svh` with min/max variants).
